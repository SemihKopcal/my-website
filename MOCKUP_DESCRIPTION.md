## 🎨 Color Application Examples

### Buttons
- **Primary**: Teal background (#14B8A6), white text
- **Secondary**: Transparent, teal border, teal text → fills on hover
- **Disabled**: Gray, reduced opacity

### Cards
- **Light Mode**: White background, light gray border
- **Dark Mode**: Slate gray background (#1E293B), darker border
- **Hover**: Border becomes teal, lifts up 8px

### Text
- **Headings**: Full contrast text color
- **Body**: 85-90% opacity for softer look
- **Links**: Teal with underline animation

### Badges/Pills
- **Light Mode**: Teal background, white text
- **Dark Mode**: Transparent with teal border, teal text → fills on hover

---

## 🎬 Interaction Flows

### Navigation Click
1. User clicks nav link
2. Smooth scroll to section (1s duration)
3. Link underline animates left-to-right

### Project Card Hover
1. Card lifts 8px upward
2. Shadow intensifies
3. Border color changes to teal
4. Image scales 110%
5. All transitions: 0.3s ease

### Skill Badge Click/Hover
1. Badge lifts 3px
2. Shadow appears/intensifies
3. Background color darkens slightly
4. Transition: 0.15s ease

### Form Submit
1. Button shows loading state (disabled)
2. Form submits
3. Success/error message fades in
4. Message auto-dismisses after 3s

---

**Design Philosophy**: Clean, professional, performance-focused with subtle but delightful interactions.
